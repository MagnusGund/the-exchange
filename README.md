# Incremental Industrialist

A new planet mod for Factorio Space Age that adds an incremental game layer. Travel to The Exchange, import materials, fulfill orders, earn credits, and unlock permanent factory-wide bonuses through Commerce Science research.

## Overview

Incremental Industrialist introduces **The Exchange** - a new planet in the Space Age galaxy. This ancient automated trading platform offers a unique economic mini-game where you:

1. **Import** raw materials from trade chests
2. **Process** them into products
3. **Export** products to fulfill orders
4. **Earn credits** from completed orders
5. **Purchase Exchange Data** with credits
6. **Craft Commerce Science** for infinite research
7. **Unlock global bonuses** that apply to ALL your planets

## The Exchange Planet

- **No hazards** - focus on building and optimizing
- **No drop pod restrictions** - bring your legendary gear
- **Small ore patches** - bootstrap materials to get started
- **Always daytime** - maximum solar power efficiency

## Core Systems

### Import Chests
- Generate configured materials automatically
- Output directly to adjacent belts
- Upgradeable import rate and material tiers

### Export Chests
- Accept products from belts
- Automatically match items to active orders
- Consume items and award credits

### Order System
- Multiple concurrent orders to choose from
- Size tiers: Small → Medium → Large → Huge → Massive
- Complete orders to unlock larger versions with better rewards
- Reroll orders for a credit cost

### Exchange Data & Commerce Science
- **Exchange Data**: Purchase with credits (scaling cost)
- **Data Matrix**: Craft from Exchange Data + circuits
- **Commerce Science Pack**: Craft from Data Matrix + advanced components
- Use for infinite research with global bonuses

### Infinite Research (Global Bonuses)
- Assembler Speed +2% per level
- Mining Productivity +2% per level
- Lab Research Speed +2% per level
- Robot Cargo Capacity +1 per level
- Robot Speed +3% per level
- Inserter Stack Size +1 per level

## Getting Started

1. Research **Planet Discovery: The Exchange** (requires Space Science)
2. Travel to The Exchange
3. Research **Trade Protocols** on the planet
4. Build Import Chests and configure materials
5. Build Export Chests and process products to fulfill orders
6. Use earned credits to purchase Exchange Data
7. Research **Commerce Science** to unlock infinite upgrades
8. Return anytime to scale up your operations!

## Requirements

- Factorio 2.0+
- Space Age DLC

## Installation

Copy this folder to your Factorio mods directory:
- Windows: `%APPDATA%\Factorio\mods\incremental-industrialist`
- Linux: `~/.factorio/mods/incremental-industrialist`
- macOS: `~/Library/Application Support/factorio/mods/incremental-industrialist`
